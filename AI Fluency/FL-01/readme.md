# FL-01: AI Workflow Audit

This repository contains my submission for the **General AI Fluency (FL-01)** assignment as part of the FlyRank Internship program.

## Project Overview
This project maps my recurring professional and academic tasks to determine the optimal level of AI integration (Collaboration, Delegation, or Automation). It also includes evidence of my environment setup, including my Claude Project configuration and Anthropic Academy certifications.

## Contents
- **`index.html`**: The main workflow audit dashboard.
- **Images**: Screenshots documenting my Claude Project instructions and course completion.

## Tasks & Classifications
The workflow audit categorizes tasks into four distinct buckets:
1. **Just me**: Tasks requiring critical human judgment or personal context.
2. **Collaborate with AI**: Tasks where AI assists, but human oversight is central.
3. **Delegate to AI**: Tasks where AI performs the bulk of the work.
4. **Fully automate**: Tasks handled entirely by AI scripts.
